# Instalación en el móvil 

1.- Tener iniciada vuestra sesión de google en Quetta, para que la herramienta tenga acceso a los datos que procesa

1.- Descargar el navegador Quetta del Play Store https://play.google.com/store/apps/details?id=net.quetta.browser

2.- Abrir esta URL desde el navegador Quetta, copiadla y pegarla en el navegador _https://github.com/Javiercb90/La-Union-Iberica-Tool_

3.- Descargar la herramienta e instalarla en el navegador Quetta, siguiendo las instrucciónes de las siguientes imágenes
